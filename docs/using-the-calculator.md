# Using the calculator

## Number and operation buttons

Enter the first number, choose an operation, enter the second number, and select **=**.

| Button | Purpose | Example |
|---|---|---|
| `+` | Add | `7 + 3 = 10` |
| `−` | Subtract | `10 − 4 = 6` |
| `×` | Multiply | `6 × 5 = 30` |
| `÷` | Divide | `20 ÷ 4 = 5` |

If you select another operation before **=**, the calculator completes the current operation and continues with the new result.

## Decimal numbers

Select the decimal point once while entering a number. A second decimal point in the same number is ignored.

Example: `2.5 × 4 = 10`.

## Negative numbers

Select **±** to change the sign of the number on the display.

- `8` becomes `-8`
- `-8` becomes `8`
- `0` stays `0`

To calculate `5 − 9`, you can enter the subtraction normally. The calculator displays `-4`.

## Percentages

The **%** button divides the displayed number by 100.

| Input | Result |
|---|---:|
| `25 %` | `0.25` |
| `150 %` | `1.5` |
| `8 %` | `0.08` |

!!! info "How this percentage button works"
    It converts a percentage to its decimal form. It does not automatically calculate “25% of another number.” To find 25% of 80, enter `25 % × 80 =`.

## Clear the calculator

**Clear** removes the current number, stored operation, and expression. It restores the initial `0` display.

## Continue a calculation

After selecting an operation, the operation button is highlighted. Enter the next number and either:

- select **=** to finish, or
- select a different operation to calculate the current step and continue.
